REGISTRATION STEPS

To register for ROHP, you must be a candidate for the Regents' and Chancellor's Scholarship. We will cross-check your name with a list from the scholarship office. Please register for a program through one of the forms linked below. About 5 days before your registered program date, program information and specifics will be sent to the email address you submitted on the form.

Step 1

Waivers
  • Please find the required event waivers linked below. You will need to fill out and sign these forms, and then scan and upload the signed copies when you sign up through the forms linked below. If you are attending the Mar. 15 program, you'll need to fill out the waiver for virtual program attendees. If you are attending one of the other programs, fill out the waiver for in-person program attendees.

REGISTRATION FORM

  • Below are the dates to each ROHP night. Each date links to an form that will allow you to register for the night of your choosing. Please only register for one night; if you have registered for more than one program, we will have to cancel all registrations under your name. If you were nominated for the Regents' and Chancellor's Scholarship and received an invitation to interview for the Scholarship, the first four programs are open for registration.

    The Mar 15th program is a fully virtual program and the zoom link will be sent to the email used to register (which should be connected to your zoom account). The Apr. 18th-19th Program is for Scholarship Recipients only, but will contain the same programming as prior dates. If you have already attended one of the first four programs, please do not sign up for the Apr. 18th-19th program. If a form requires a password, that means we have filled up all of the guest spots for that program and you may register for the waitlist here. Make sure to fill out the waitlist form only once single and indicate which dates you would like to be waitlisted for if time if you would like to be on the waitlist for multiple nights.
  • To access the registration forms, please click on one of the bears with the program date you desire to attend.
